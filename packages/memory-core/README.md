# memory-core

Shared memory abstractions for learner and training state.

## Purpose

Provide durable memory building blocks for long-term skill development.

## Memory Layers

### 1. Learner Profile
Long-term learner facts:
- active domains
- current strengths
- recurring weaknesses
- recent trend

### 2. Training Memory
Session-derived memory:
- recent scenarios
- diagnosis history
- recurring failure patterns
- effective interventions

## Important Boundary

This package stores and structures memory.
It does not decide routing or scoring.

# gp-visit-jp-runtime

Minimal second-domain runtime used to prove the shared runtime contract against a
non-MR domain.

## Scope

- text-only GP communication spike
- 2 scenarios
- 3 domain-owned subskills
- deterministic turn/review/progress logic
- in-memory storage only

## Non-Goals

- production persistence
- model-backed judge/coach behavior
- shared package extraction before the spike proves the need

# Skill Model

## Core Subskills
- preparation
- opening
- profiling
- scientific_delivery
- need_discovery
- objection_handling
- closing_followup

## Weighting
- `preparation`: 0.10
- `opening`: 0.14
- `profiling`: 0.14
- `scientific_delivery`: 0.20
- `need_discovery`: 0.18
- `objection_handling`: 0.14
- `closing_followup`: 0.10

## Score Range
Each subskill: 0-5
Weighted overall score: 0-100

## Score Bands
- `critical_gap`: 0-39
- `emerging`: 40-59
- `functional`: 60-74
- `strong`: 75-89
- `excellent`: 90-100

## Evaluation Rules
- Score observable behavior, not inferred intent.
- Missing evidence scores low instead of being guessed upward.
- Compliance remains a separate tagging pipeline and does not replace subskill scoring.
